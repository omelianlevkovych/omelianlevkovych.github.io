My Role

During this project I oversaw how Linq works and improved my Unit Testing & TDD skills.

Project Difficulties

This project really teached me test driven development approach. Some struggles were faced during the creation of proper unit tests before even implemention some of the LINQ operators. For istance, in order to verify that operator is deferred I have spent some time thinking how exactly such behaviour can be proved. Also it was a bit unusual to code in a bit of functional programming way. Lastly, at first it was a little demanding to understand some LINQ hacks in JIT (e.g SZArrayHelper class).

My Solution

I have read many articles and some books to dive a bit deeper into TDD and functional programming. Following of the information with which I was provided I managed to step over my difficulties. Further, to get a basic understanding how JIT hacks in LINQ works, I have read a bunch of .NET Framework source code and asked few questions in StackOverflow.
